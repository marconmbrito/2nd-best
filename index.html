Hey! I'm Marco Brito, a self-taught front-end developer, based in Stockholm, Sweden. A clean, tidy and beautiful code is a priority for me, as well as delivering a modern design with the best UX possible. First and foremost, developing is solving problems in the simplest and most efficient way to the advantage of all stakeholders.

  • - Mobile first //

    This ultimately means that your website will be carefully crafted in a way that it will work across multiple devices, no matter the screen size or resolution. Start small and scale up.

  • User Interface //

    Design, layout and colors. These are things that might ruin and level-up a user experience. I take this in ultimate importance.

  • User Experience //

    My approach is with a deep and thourough thought on this matter. Research and planning are particularly important at this point.

  • DRY Code //

    DRY stands for 'Don't Repeat Yourself', and I do my absolute best in having my code as clean as possible.

  • Sass //

    When it comes to keep a clean and tidy code, Sass is my best friend. Writting organized and maintanable CSS has never been easier and more fun.

  • jQuery //

    When it comes to interactivity, jQuery handles it. This is the tool I'll use to make your website fun and interactive, thus improving your UX.

The journey so far

My coding journey has been filled with hard work and lots of discovery. Here you can see a little bit of how I got into this fascinating world of coding.

May 2017

It was on the 5th when I first stumbbled on coding. "It is not for me" - I thought. But let's give it a week and see how it goes...

June 2017

... It did not take a week. 2 days after and I couldn't do anything other than coding. One month later and I was coding every day. I'd fallen in love.

August 2017

Almost 3 months later and I had already built some small things, nothing put online. But the passion was alive and I knew what I wanted to do for the rest of my life...

October 2017

This was the month where I started building this website from scratch. From planning, going through research, wireframe, design, copy and code, I had a lot of fun.

November 2017

This is the momment of launch. First project to actually go online. Small step for the internet, big step for me :) More to come, a life of code.
